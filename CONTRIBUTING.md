# Contributing to OpenMedSim3D

Thank you for helping improve OpenMedSim3D.

Our goal is to create an open-source library of high-quality 3D-printable medical simulation models that support healthcare education worldwide.

## Ways to Contribute

You can contribute by:

- Uploading new medical simulation models
- Improving existing models
- Reporting issues with models
- Sharing print settings
- Providing assembly instructions
- Testing models and giving feedback
- Suggesting new simulation projects

## Submission Requirements

Each model should include:

### Required Files

- STL files
- At least one image of the printed model
- A README.md file

### Recommended Files

- STEP or CAD files
- Print settings
- Assembly instructions
- Material list
- Cost estimate

## Model Documentation

Each model README should include:

- Model name
- Educational purpose
- Procedure being simulated
- Materials required
- Printer used
- Print settings
- Assembly instructions
- Known limitations

## Quality Standards

Models should:

- Be printable on commonly available printers
- Be appropriately documented
- Serve a clear educational purpose
- Be safe for educational use
- Comply with repository licensing requirements

## Reporting Problems

If you encounter an issue:

1. Open an Issue.
2. Describe the problem.
3. Include photos if possible.
4. Include printer settings used.

## Code of Conduct

Be respectful and collaborative.

OpenMedSim3D welcomes contributions from educators, students, clinicians, engineers, researchers, and makers worldwide.
